INSTITUTE OF SCIENCE AND TECHNOLOGY
Course Syllabus - Fall 2024

Course: CS401 - Cybersecurity Fundamentals
Instructor: Prof. Sarah Johnson
Credits: 4

Course Description:
This course provides a comprehensive introduction to cybersecurity concepts, including 
network security, cryptography, web security, and ethical hacking fundamentals.

Topics Covered:
1. Introduction to Cybersecurity
2. Network Security
3. Cryptography Basics
4. Web Application Security
5. Common Vulnerabilities (OWASP Top 10)
6. Penetration Testing
7. Incident Response
8. Security Tools and Techniques

Grading:
- Assignments: 30%
- Midterm Exam: 25%
- Final Project: 30%
- Participation: 15%

Required Materials:
- "Web Application Security" by Andrew Hoffman
- "The Web Application Hacker's Handbook" by Dafydd Stuttard
